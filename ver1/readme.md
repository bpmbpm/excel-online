### function
Полное отображение URL-адреса с параметром file — при вводе URL-адреса отображается полная ссылка ниже (например, https://bpmbpm.github.io/excel-online/ver1/index.html?file=<url>), которую можно использовать для непосредственного открытия файла. https://github.com/bpmbpm/excel-online/pull/6

### example
https://bpmbpm.github.io/family-tree/ver6/tree.xlsx (лежит в https://github.com/bpmbpm/family-tree/blob/main/ver6/tree.xlsx но если оттуда, то будет CORS)
